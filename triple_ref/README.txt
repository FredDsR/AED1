O programa triple_ref.c remete a pática de referência tripla para manipulação de listas encadeadas. O conteúdo está explicado no vídeo cujo o link está explicitado abaixo. O progama armazenado aqui tem fins de estudo e análise do código e foi retirado de https://www.youtube.com/watch?v=0ZEX_l0DFK0&ab_channel=Computerphile


- Estrutura do diretório:
    |_ triple_ref.c - Arquivo fonte do programa.
    |_ README.txt   - Documentação do programa.
    |_ Makefile     - Arquivo com comandos úteis para a execução do comando.


- Como rodar:
    Para rodar o programa desenvolvido basta compilar com o comando "make compile" e, em seguida, rodar o comando "make run".
