- Enunciado:
    Faça um programa que armazena nomes. O programa deve armazenar todos os nomes na mesma string. O tamanho da string deve crescer e diminuir conforme nomes forem colocados ou removidos. Não pode ter desperdício de memória.
    Faça o seguinte menu:
        1) Adicionar nome 
        2) Remover nome
        3) Listar
        4) Sair


- Estrutura do diretório:
    |_ exercício2.c - Arquivo fonte do programa.
    |_ README.txt   - Documentação do programa.
    |_ Makefile     - Arquivo com comandos úteis para a execução do comando.


- Como rodar:
    Para rodar o programa desenvolvido basta compilar com o comando "make compile" e, em seguida, rodar o comando "make run".
