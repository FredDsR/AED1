- Enunciado:
    Usar IDE com debugger. Faça uma gravação curta de você criando um projeto "Olá Mundo" em C no Visual Code e usando o debug com um breakpoint no printf. 
    GitHUB:
        -Comitar/upload no final de cada semana todos os seus exercícios para lá 
        - Crie pastas para cada exercício 
        - Colocar o link para o seu repositório no pacote no e-aula da lista de exercícios 1


- Estrutura do diretório:
    |_ exercício1.c - Arquivo fonte do programa.
    |_ README.txt   - Documentação do programa.
    |_ Makefile     - Arquivo com comandos úteis para a execução do comando.


- Como rodar:
    Para rodar o programa desenvolvido basta compilar com o comando "make compile" e, em seguida, rodar o comando "make run".
